SUMMARY
=======

Project Structure:
- Monorepo using pnpm workspaces and Turborepo
- 2 applications: Discord bot & Next.js web application
- 5 shared packages: api, db, env, messaging, typescript-config

Applications:
1. apps/discord/ - Discord bot with commands (admin & public) and scheduler
2. apps/nextjs/ - Next.js web application with dashboard and API routes

Packages:
1. packages/api/ - Shared API layer with tRPC routers, repositories, and services
2. packages/db/ - Database package with Prisma ORM
3. packages/env/ - Environment variable management
4. packages/messaging/ - Message broker service
5. packages/typescript-config/ - Shared TypeScript configuration

Key Technologies:
- TypeScript
- Discord.js
- Next.js (App Router)
- tRPC
- Prisma ORM
- pnpm (package manager)
- Turborepo (monorepo tool)
